# XPort Studio

XPort Studio is a premium web application designed to help users quickly build custom datasheet templates and export them into multiple professional formats including PDF, Word (DOCX), and Excel (XLSX).

Built with a luxury-inspired Gold + Black UI theme, parallax motion, and a guided multi-step workflow ‚Äî XPort Studio allows you to design once and export everywhere.

---

## ‚ú® Features

- Multi-format export:
  - PDF
  - DOCX (Microsoft Word)
  - XLSX (Microsoft Excel)
- Step-by-Step Guided Wizard:
  1. Title & file name
  2. Table structure (columns + row count)
  3. Format selection
  4. Preview & export
- Custom column colors using **Header Color Picker**
- Stunning premium UI with:
  - Neo-glass design
  - Parallax hover animation
  - Smooth transitions (Framer Motion)
- Print-friendly result templates
- Fully responsive (Desktop + Tablet + Mobile)
- Built for deployment on Vercel
